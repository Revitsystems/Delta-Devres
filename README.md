# Delta Devres Website

Official marketing website for Delta Devres International.

## Tech Stack

- HTML5
- CSS3 (Custom + Bootstrap 5 via CDN)
- Vanilla JavaScript
- No backend server required

## Project Structure

- index.html – Home page
- about.html - About page
- our-work.html - Our Work page
- project.html - Project page
- legal.html - Privacy and copyright information
- assets/css/Style.css – Global styles
- assets/js/Script.js – Navigation, form handling, UI logic
- assets/images/ – All site images

## Running the Project

1. Download or clone the project
2. Open `index.html` in a browser
   OR
3. Serve using any static server (recommended)

## External Dependencies

- Bootstrap 5 (CDN)
- Google Fonts
- Web3Forms API (contact form)
- Brevo – Newsletter subscription via embedded form (no custom JS logic)
- Google Maps Embed

## Deployment

Deployed on:

- Vercel
- GitHub Pages
